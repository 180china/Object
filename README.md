# Object  4.0.0
HTML5 2DProject 

Mobile Html5 frameworks
Based on pixi.js https://github.com/pixijs/


## Usage

Installation

```bash
npm install
```

开发：
```bash
gulp serve
```

Build

```bash
grunt
```

uglify Libs

```bash
grunt libs
```
