# Object  5.0.0
HTML5 2DProject 

Mobile Html5 frameworks
Based on pixi.js https://github.com/pixijs/

预览html页面地址:http://htmlpreview.github.com/. 


## Usage

Installation

```bash
npm install gulp -g
npm install 
```

dev：
```bash
gulp dev
```

Build
```bash
gulp libs
gulp minifyjs
```
